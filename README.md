# Calendar Application (Frontend Assignment)

This project is a monthly calendar web application built as part of a frontend interview assignment. The goal was to create a user-friendly calendar interface similar to Google Calendar, with support for event display and smooth month navigation.

# Tech Stack

React – component-based UI development
Vite – fast build tool and development server
Vanilla CSS – custom styling without external UI libraries
dayjs – lightweight date manipulation library

# Features

Displays the current month and year by default
Previous / Next month navigation
Calendar grid with weekday headers (Sun–Sat)
Current date highlighted for better UX
Events loaded from a static JSON file
Events displayed on their respective dates
Supports multiple events on the same day using color-coded event blocks
Clean, modern, and minimal UI inspired by Google Calendar–style layouts

## React Compiler
